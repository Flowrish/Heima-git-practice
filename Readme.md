下图是常见命令，标红部分为本节补充的

| 操作              | 命令                            |
| ----------------- | ------------------------------- |
| 查看分支          | git branch                      |
| 创建分支          | git branch 新分支名             |
| 切换分支          | git checkout 分支名             |
| **创建+切换分支** | git checkout –b 新分支名        |
| 合并分支          | git merge 分支名                |
| 删除分支          | git branch –d 分支名            |
| **强制删除分支**  | git branch –D 分支名            |
| 重命名分支        | git branch –m 老分支名 新分支名 |
